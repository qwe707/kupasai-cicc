## 二、推荐 Top 3

### 推荐 1：工程设计规范智能问答（首推）
- **核心思路**：RAG（检索增强生成）+ 规范知识图谱
- **数据**：公开建筑/市政设计规范 PDF（GB、JGJ 系列等）
- **技术栈**：向量库（Qdrant / Milvus）+ 大模型（开源/调用）+ FastAPI + 简单 Web UI
- **创新点**：规范条款引用追溯、多规范交叉冲突检测、可视化条款图谱
- **演示效果**：问答交互直观，截图/视频好出
- **风险**：低

### 推荐 2：标书自动化生成智能体
- **核心思路**：多 Agent 编排（需求理解 Agent → 素材检索 Agent → 章节生成 Agent → 合规审查 Agent → 排版 Agent）
- **数据**：公开历史标书模板 + 行业素材
- **技术栈**：LangGraph / 自研 Agent 框架 + RAG + 模板引擎
- **创新点**：标素材自动分解、标准映射、多 Agent 协同
- **演示效果**：从招标书到完整投标书的端到端 demo，视觉冲击强
- **风险**：中（流程复杂，时间偏紧）

### 推荐 3：检索增强的造价领域对话问答
- **核心思路**：与推荐 1 类似，垂直在造价定额、清单计价
- **创新点**：定额条款检索、计价公式自动推导
- **风险**：低-中

- [x] **已锁定赛题**：**基于多模态大模型赋能的城市轨道交通保护区安全管控**（自定义命题，建交赛道）

- [x] **决策人**：团队全体
- [x] **决策依据**：

  2. 技术差异化 — YOLO 视觉检测 + MLLM 语义研判**双模融合**，相比纯 RAG / 纯 CV 方案有创新性

  5. 展示效果 — 实时视频识别 + 告警卡片 + 大屏，决赛现场演示张力强
  6. 评审契合 — 三维度（创新性 / 适配度 / 落地潜力）全部强对齐
